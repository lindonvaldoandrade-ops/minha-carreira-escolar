🖥️ Informática


💡 O que é informática?

📘 Informática é usar o computador para trabalhar com informações.
Podemos escrever, ouvir música, assistir vídeos e falar com pessoas pela internet.


🖥️ Um computador tem partes importantes:

  1. 👀 Monitor → onde vemos as imagens
  2. ⌨️ Teclado → onde digitamos as letras
  3. 🖱️ Mouse → para clicar e mover coisas na tela
  4. ⚙️ CPU → faz o computador funcionar

📦 Dentro do computador existem programas (softwares), como o Word, o Paint e o navegador da internet.
