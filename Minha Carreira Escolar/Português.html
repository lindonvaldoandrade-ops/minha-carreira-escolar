🧠 Conteúdo: Interpretação de Texto

📝 Explicação Simples:

Interpretar um texto é entender o que ele está dizendo. Você lê um texto e depois responde
perguntas para mostrar que entendeu. Não é preciso decorar. O importante é prestar atenção
nas palavras e no que elas querem dizer.

💡 Dica: Leia devagar, com calma. Se precisar, leia mais de uma vez.

✅ Exemplo de Texto Curto:

Texto:

João tem um cachorro chamado Max. Todos os dias, João passeia com Max no parque.
Max gosta de correr atrás da bola e brincar com outros cachorros. João se diverte muito com Max.


✍️ Exemplo de perguntas com ajuda visual e simples:


1º Quem é o dono do cachorro?

( ) Pedro
(✔) João
( ) Lucas

2º Qual é o nome do cachorro?

( ) Rex
( ) Totó
(✔) Max

3º Onde João passeia com Max?

( ) No shopping
(✔) No parque
( ) Na escola

4º O que Max gosta de fazer?

☐ Dormir
☑ Correr atrás da bola
☐ Comer


🧩 Atividade prática (para o aluno fazer):


📖 Leia o texto:

Ana tem uma gatinha chamada Lili. Lili é branca e tem olhos azuis.
Ana gosta de brincar com Lili no tapete da sala. À noite, Lili dorme na cama de Ana.


✏ Responda:


1º De que cor é a Lili?

( ) Preta
( ) Laranja
( ) Branca

2º Ana brinca com Lili no...

( ) quintal
( ) tapete da sala
( ) sofá br


💡 Dicas para facilitar o aprendizado:


1º Use imagens do texto (ex: desenho da Lili, do parque, etc.).

2º Leia o texto junto com o aluno, se necessário.

3º Reforce com perguntas orais antes da escrita.

4º Evite textos muito longos ou perguntas com muitas opções.
